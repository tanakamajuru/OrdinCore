1. **Capture:** Screenshot, browser console logs, network response.

4. **Report (if cannot fix):** If the issue requires architectural changes or clarification, log it in an error report with:
   - Test Step ID
   - Description of failure
   - Expected vs Actual
   - Relevant spec reference
   - Suggested remediation
